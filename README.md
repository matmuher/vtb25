# ОткатЪ

### Сервис по анализу лучших категорий кешбека на следующий месяц + выбор оптимальной карты для оплаты в магазине

##### Описание

Сервис анализирует ваши транзакции за последние несколько месяцев (полученные с помощью OpenBanking API), анализирует их, а также доступные
на картах разных банков категории кешбеков, а затем предлагает оптимальные категории на следующий месяц, чтобы получить от них максимальную
выгоду.

Также сервис подсказывает вам, какую карту выбрать при совершении покупки в магазине, чтобы получить максимальный кешбек, и пересылает
на приложение соответствующего банка

##### Как раскатить на сервере

На данный момент:
Необходимы файлы Cashbacks.xlsx и credentials.json в директории backend, из которых читаются кешбеки банков и креды для получения access_token's банков
Пример заполнения Cashbacks.xlsx:
| bank  | category  | percent | max_categories_in_bank | category_limit | bank_limit |
|-------|-----------|---------|------------------------|----------------|------------|
| Sbank | Hair Cut  | 5       | 3                      |                | 3000       |

Формат credentials.json:

[
    {
        "bank_name": "abank",
        "client_id": "team089",
        "client_secret": "your_client_secret"
    }
]

Команда для запуска backend части:

`uvicorn backend:app --reload &`

Запуск фронтенда описан в README.md в директории frontend

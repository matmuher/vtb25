# How to start?

steps to build:

`$ npm install`
`$ npm run dev -- --host`

in console you will see something like this, but with your devices' IPs:

```
> frontend@0.0.0 dev
> vite --host

Port 5173 is in use, trying another one...

  VITE v7.1.12  ready in 264 ms

  ➜  Local:   http://localhost:5174/
  ➜  Network: http://192.168.0.105:5174/

```

# How to check on phone?

Connect your phone to the same network (same WiFi). In browser open "Network" address from YOUR message.
